---

Available commands:
- `create`: Create a new torrent file.
- `check`: Verify the integrity of content against a torrent file.
- `inspect`: Inspect the metadata of a torrent file.
- `modify`: Modify an existing torrent file.
- `update`: Update mkbrr to the latest version.
- `version`: Show the current version of mkbrr.
