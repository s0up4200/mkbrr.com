## Usage

```bash
  mkbrr check <torrent-file> <content-path> [flags]
```

## Arguments

-   `<torrent-file>`: Path to the `.torrent` file. (Required)
-   `<content-path>`: Path to the directory or file containing the data to check. (Required)
