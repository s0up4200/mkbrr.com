# `mkbrr check`

Verify the integrity of content against a torrent file. Checks if the data in the specified content path (file or directory) matches the pieces defined in the torrent file.

```bash
Usage:
  mkbrr check <torrent-file> <content-path> [flags]
```

## Arguments

-   `<torrent-file>`: (Required) Path to the `.torrent` file.
-   `<content-path>`: (Required) Path to the directory or file containing the data.

## Flags

<ParamField path="--verbose, -v" type="boolean" default="false">
  Show list of bad piece indices.
</ParamField>

<ParamField path="--quiet" type="boolean" default="false">
  Reduced output mode (prints only completion percentage).
</ParamField>
