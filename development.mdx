## Prerequisites

- **Go:** Version 1.23 or higher (as specified in `go.mod`).
- **Git:** For cloning the repository.
- **Make:** For using the provided build and test commands.

## Building from Source

Clone the repository and build the binary:

```bash
git clone https://github.com/autobrr/mkbrr.git
cd mkbrr
make build
```

<Info>
  The built binary will be placed in the `build/` directory.
</Info>

## Common Development Tasks

- **Build:** `make build`
- **Test with race conditions:** `make test-race`
- **Lint:** `make lint`
- **Clean:** `make clean`

<Info>
  For a full list of available commands, run `make help` or check the `Makefile`.
</Info>
